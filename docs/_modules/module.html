Module Module

Alfred Botman

Compatibility :

Description

This module allows to list, load and unload modules.
This module is mostly a POC, but could be used for example to update an alfred module without restart alfred.

Configuration

There isnt any configuration to be done for this module.

Commands

Loading and unload modules might be handy for debugging purposes, or if only a few modules gets updated.
  • List modules
  • Load module
  • Unload module

Alfred Botman

This command allows you to load a module identified by its name (not path).
(17:04:28) guillaume.friloux@botnet.master.com: .module load gdb
(17:04:29) botman: Module gdb successfully loaded.

Alfred Botman

Unloads module by name. 
(17:02:34) guillaume.friloux@botnet.master.com: .module unload gdb
(17:02:34) botman: Module gdb successfully unloaded.
