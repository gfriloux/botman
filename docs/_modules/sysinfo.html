Sysinfo Module

Alfred Botman

Compatibility:

Description

This module fetch various system informations, and stores them on Alfred.
You can also abuse this module to create access to various system commands.

Configuration

  • Alfred
  • Botman

{
   "vars" : [
      "raid_status",
      "disk_full"
   ]
}
