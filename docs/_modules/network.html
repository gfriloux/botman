Network Module

Botman

Description

This module returns the network configuration.
This module only works on Windows at the moment.


Configuration

This module does not need any configuration file.

Query botmans through Alfred

It is possible to make alfred able to query botmans for you, but editing the configuration file from the rewrite module.
You can add the following JSON object as a new rule of the rewrite module : 

{
   "name" : ".network",
   "filter" : ".network*",
   "rule" : "sed -r 's/^.network (.*)/.spam \\1 .network/'",
   "description" : ".network "
}
